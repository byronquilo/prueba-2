INTRODICCIÒN

HTML se utiliza pata crear paginas web en internet es el lenguaje con el que escriben la mayoría de paginas web que solemos encontrar en el internet Creado por Tim Berners Lee en el CERN (Laboratorio nuclear de Ginebra)era utilozado para la compartición de documentos científicos en la comunidad académica es un lenguaje de marcado muy pequeño la cual solo tenia 20 etiquetas las cuales perduran 13. Debido ala crecimiento de la web surgió la necesidad de crear un estándar tanto que programadores y navegadores pidieran basarse en una sola norma de HTML

VERCIÒN DE HTML

VERSION 2.0 En 1995 se publica el estándar HTML 2.0. A pesar de su nombre, HTML 2.0 es el primer estándar oficial de HTML, es decir, el HTML 1.0 no existió como estándar. HTML 2.0 no soportaba tablas.Se simplificaba al máximo la estructura del documento para agilizar su edición, donde la declaración explícita de los elementos body, html y head es opcional.
VERSION 3.2 La versión HTML 3.2 se publicó en 1997 y es la primera recomendación de HTML publicada por el W3C (Consorcio internacional). Esta revisión incorporó los últimos avances de las páginas web desarrolladas hasta 1996, como applets de Java y texto que fluye alrededor de las imágenes.
HTML 4.O1 La última especificación oficial de HTML se publicó en diciembre de 1999 y se denomina HTML 4.01. Desde la publicación de HTML 4.01, el W3C se centró en el desarrollo del estándar XHTML. Por este motivo, en el año 2004, las empresas Apple, Mozilla y Opera mostraron su preocupación por la falta de interés del W3C en HTML
HTML 5 El consorcio internacional W3C, después de una evolución de varios años, liberó el HTML 5 como estándar oficial a finales de octubre de 2014. HTML 5 incorpora nuevos elementos no contemplandos en HTML 4.01. Hay diversos cambios respecto a HTML 4.01. Hay nuevas etiquetas, se introduce la posibilidad de introducir audio y video de forma directa en la web sin necesidad de plugins o complementos en los navegadores, y otras novedades. El W3C irá lanzando progresivamente nuevas evoluciones del HTML 5.
